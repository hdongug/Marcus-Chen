# Marcus Chen 포트폴리오 - 디자인 아이디어

## 선택된 디자인 철학: 테크 미니멀리즘 + 네온 악센트

### 디자인 무브먼트
**테크 미니멀리즘** - 기술 업계의 세련된 미학과 극도의 단순함을 결합한 현대적 접근

### 핵심 원칙
1. **극도의 단순함**: 불필요한 요소 제거, 본질만 남김
2. **명확한 계층구조**: 타이포그래피와 공간으로 정보 우선순위 표현
3. **기술적 우아함**: 코드의 아름다움을 시각적으로 표현하는 디자인 선택
4. **상호작용 중심**: 마우스 오버, 클릭 등에 반응하는 미묘한 애니메이션

### 색상 철학
- **배경**: 진한 검정(`#0a0a0a`) - 깊이감과 집중력 제공
- **주요 텍스트**: 밝은 슬레이트 그레이(`#e8e8e8`) - 눈 피로 최소화
- **악센트**: 네온 그린(`#00ff41`) - 기술적 에너지와 생동감
- **보조 악센트**: 슬레이트 그레이(`#64748b`) - 부차적 정보 표현
- **강조**: 네온 그린의 투명도 변화로 깊이 표현

### 레이아웃 패러다임
- **비대칭 구조**: 좌측 콘텐츠, 우측 여백 활용
- **수직 리듬**: 일관된 간격으로 시각적 안정감 제공
- **그리드 기반**: 프로젝트와 스킬은 카드 그리드로 표현
- **플로팅 요소**: 일부 섹션은 배경에서 떠있는 느낌

### 시그니처 요소
1. **네온 그린 보더**: 주요 섹션과 카드의 좌측 또는 상단 보더
2. **코드 스타일 텍스트**: 모노스페이스 폰트로 기술 요소 강조
3. **그래디언트 배경**: 검정에서 진한 회색으로의 미묘한 그래디언트

### 상호작용 철학
- **호버 효과**: 카드가 미묘하게 위로 올라오고 그린 글로우 추가
- **클릭 반응**: 스케일 애니메이션으로 즉각적인 피드백
- **스크롤 트리거**: 요소가 화면에 들어올 때 페이드 인 + 슬라이드

### 애니메이션 가이드라인
- **진입 애니메이션**: 0.6초 ease-out으로 부드러운 페이드 인
- **호버 상태**: 0.2초 ease-in-out으로 빠른 반응
- **로딩 상태**: 네온 그린 색상의 펄싱 애니메이션
- **전환**: 0.3초 ease-in-out으로 페이지 간 부드러운 전환

### 타이포그래피 시스템
- **디스플레이**: `Space Mono` (Bold) - 헤더, 제목 (기술적이고 강렬함)
- **본문**: `Inter` (Regular, Medium) - 본문, 설명 (가독성 높음)
- **악센트**: `JetBrains Mono` (Regular) - 코드, 기술 용어 (개발자 친화적)

### 계층 구조
- H1: `Space Mono` 48px Bold, 네온 그린
- H2: `Space Mono` 32px Bold, 밝은 슬레이트 그레이
- H3: `Space Mono` 24px Medium, 밝은 슬레이트 그레이
- Body: `Inter` 16px Regular, 밝은 슬레이트 그레이
- Small: `Inter` 14px Regular, 슬레이트 그레이
- Code: `JetBrains Mono` 14px Regular, 네온 그린

## 디자인 결정 사항
- **다크 모드 기본**: 기술 업계 표준이자 개발자 친화적
- **네온 그린 선택**: 기술, 에너지, 미래지향적 느낌 전달
- **미니멀한 공간 활용**: 정보 과부하 방지, 집중력 유지
- **마이크로 인터랙션**: 사용자 참여도 증가, 프리미엄 느낌
